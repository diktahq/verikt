archway serves two audiences with the same tool.

**[For Engineers →](/for/engineers/)**
You use AI agents to write code. archway makes sure every session starts with your architecture — layer boundaries, capability stack, dependency rules. No more re-explaining your patterns.

**[For Engineering Leaders →](/for/engineering-leaders/)**
Your team uses AI agents at scale. archway enforces architecture consistency across every agent, every engineer, every PR — without adding review burden.

Same CLI. Same `archway.yaml`. Different problems, same solution.
