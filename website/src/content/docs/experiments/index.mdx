archway's claims are backed by experiments, not opinions. Each experiment isolates a single variable, measures the outcome, and reports what happened — including null results and invalid runs. We publish falsified hypotheses alongside validated ones.

Each experiment runs the same agent on the same task. Condition A: no guide — what most teams do today. Condition B: `archway guide` output prepended to the prompt. Same model, one variable.

## Methodology

Every experiment follows the same protocol: same agent, same task, one variable changed. Every run persists the full prompt, raw response, and `archway check` output as artifacts. [Full methodology →](/experiments/methodology/)
