## Install

<Tabs>
<TabItem label="Homebrew">

```bash
brew install diktahq/tap/verikt
```

</TabItem>
<TabItem label="Install Script">

```bash
curl -sSL https://verikt.dev/install.sh | bash
```

Works on macOS and Linux. Installs the binary and registers verikt with
your AI agents automatically.

</TabItem>
<TabItem label="From Source">

```bash
git clone https://github.com/diktahq/verikt.git
cd verikt
go build -o verikt ./cmd/verikt
```

</TabItem>
</Tabs>

## Verify

```bash
verikt --version
```

## Shell Completions

<Tabs>
<TabItem label="Bash">

```bash
verikt completion bash > /etc/bash_completion.d/verikt
```

</TabItem>
<TabItem label="Zsh">

```bash
verikt completion zsh > "${fpath[1]}/_verikt"
```

</TabItem>
<TabItem label="Fish">

```bash
verikt completion fish > ~/.config/fish/completions/verikt.fish
```

</TabItem>
</Tabs>

<Aside type="tip">
After installing completions, restart your shell or run `source ~/.bashrc` (or equivalent) for them to take effect.
</Aside>
