## Install

<Tabs>
<TabItem label="Homebrew">

```bash
brew install dcsg/tap/archway
```

</TabItem>
<TabItem label="Install Script">

```bash
curl -sSL https://archway.dcsg.me/install.sh | bash
```

Works on macOS and Linux. Installs the binary and registers archway with
your AI agents automatically.

</TabItem>
<TabItem label="From Source">

```bash
git clone https://github.com/dcsg/archway.git
cd archway
go build -o archway ./cmd/archway
```

</TabItem>
</Tabs>

## Verify

```bash
archway --version
```

## Shell Completions

<Tabs>
<TabItem label="Bash">

```bash
archway completion bash > /etc/bash_completion.d/archway
```

</TabItem>
<TabItem label="Zsh">

```bash
archway completion zsh > "${fpath[1]}/_archway"
```

</TabItem>
<TabItem label="Fish">

```bash
archway completion fish > ~/.config/fish/completions/archway.fish
```

</TabItem>
</Tabs>

<Aside type="tip">
After installing completions, restart your shell or run `source ~/.bashrc` (or equivalent) for them to take effect.
</Aside>
