`elasticsearch` wires an Elasticsearch client into your Go service with index management and search query builders.

## What You Get

- Client setup
- Index lifecycle management
- Search query builders
- Bulk operations

## When to Use

Use `elasticsearch` for full-text search, log analytics, or document indexing where SQL LIKE queries fall short. Commonly used alongside a primary database — write to Postgres, sync to Elasticsearch for search.

## Requirements & Suggestions

**Suggests:** [`observability`](/capabilities/observability/), [`docker`](/capabilities/docker/)

## Add to Your Service

```bash
archway new my-service --cap elasticsearch
# or add to an existing service:
archway add elasticsearch
```
