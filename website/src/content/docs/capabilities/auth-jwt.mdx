`auth-jwt` adds JWT authentication middleware to your Go service, validating tokens and extracting claims into the request context.

## What You Get

- Middleware that validates JWT tokens
- Claims extraction into request context
- Route-level authentication

## When to Use

Use `auth-jwt` for stateless API authentication where you don't need server-side session storage. JWTs carry claims directly in the token, making them well-suited for microservices where every service needs to verify identity independently.

## Requirements & Suggestions

**Requires:** [`http-api`](/capabilities/http-api/)

## Add to Your Service

```bash
archway new my-service --cap http-api,auth-jwt
# or add to an existing service:
archway add auth-jwt
```
