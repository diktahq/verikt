`kafka-consumer` adds a Kafka consumer to your Go service with a handler interface and graceful shutdown drain.

## What You Get

- Consumer group setup
- Message handler interface
- Graceful shutdown with drain
- Config for brokers/topics/consumer group

## When to Use

Use `kafka-consumer` for event-driven architectures and async message processing. Always pair with `graceful` shutdown — ungraceful shutdown causes rebalancing storms and potential duplicate processing.

## Add to Your Service

```bash
archway new my-service --cap kafka-consumer
# or add to an existing service:
archway add kafka-consumer
```
